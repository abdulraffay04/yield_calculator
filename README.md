# 🌾 Yield Calculator (Python Project)

## Features
- Land size input
- Crop selection
- Cost breakdown (seeds, water, fertilizer, spray, labor)
- Yield estimation
- Profit calculation

## Project Description

Smart Crop Yield & Profit Calculator is a Python-based agricultural calculator that helps farmers and students estimate crop production, total farming costs, and net profit based on land size and selected crop.

The system takes user input for land size and crop type (wheat, rice, maize) and calculates:

- Expected crop yield per acre
- Revenue based on market price
- Total farming cost (seeds, water, fertilizers, sprays, labor)
- Final profit or loss

This project demonstrates basic agricultural economics using programming logic and can be extended into an AI-based smart farming system in the future.

It is designed to help users understand farming profitability in a simple and practical way.

## Run Project

```bash
python main.py
```
